# nfc-inout-lite

入退室管理のみを行うクライアント。

ユーザー・カードの登録は[nfc-inout](https://github.com/opera7133/nfc-inout)で行います。

## ビルド

### 1. 効果音の用意

効果音をルートフォルダに `success.wav` を保存します。

### 2. .envを設定

`.env.sample`を参考に`.env`を設定します。

### 3. パッケージのインストール

`npm i`を実行してパッケージをインストールします。
